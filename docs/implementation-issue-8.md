## 実装内容

### 1. ヒーローセクション (`HeroSection.tsx`)

**機能:**

- メインビジュアル（グラデーション背景）
- キャッチコピー「写真で見て選べる、栃木の優良企業」
- CTAボタン
  - メイン: 「業者を探す」
  - サブ: 電話番号（0120-000-000）
- 信頼性指標
  - 登録企業500社以上
  - 審査済み優良企業
  - 完全無料で利用可能

**技術的特徴:**

- レスポンシブデザイン（モバイル/タブレット/デスクトップ）
- ホバーアニメーション
- アクセシビリティ対応（aria属性）

### 2. 業種タブセクション (`IndustryTabsSection.tsx`)

**機能:**

- 3つの主要業種タブ
  - 建設業（Building2アイコン）
  - 飲食業（Utensils アイコン）
  - 小売業（ShoppingBagアイコン）
- 各業種6つのサブカテゴリ表示
- タブ切り替えによる動的コンテンツ変更

**業種構成:**

| 業種   | サブカテゴリ                                     |
| ------ | ------------------------------------------------ |
| 建設業 | リフォーム、新築、外構、塗装、解体、内装         |
| 飲食業 | 居酒屋、カフェ、レストラン、ラーメン、焼肉、和食 |
| 小売業 | 家具、衣料、食品、雑貨、電化製品、書籍           |

**技術的特徴:**

- useState フックによる状態管理
- アニメーション（fade-in）
- グリッドレイアウト（PC: 6列、モバイル: 2列）

### 3. 人気業者セクション (`PopularBusinessesSection.tsx`)

**機能:**

- 業者カードグリッド表示（モックデータ6件）
- 各カードの情報:
  - 業者名、カテゴリ
  - 評価スコア（星5段階）
  - レビュー件数
  - 所在地
  - 特徴タグ
  - Instagram連携バッジ
  - 人気バッジ（featured業者）
- 「もっと見る」ボタン

**技術的特徴:**

- カードコンポーネント（BusinessCard）
- ホバーエフェクト（拡大・影）
- レスポンシブグリッド（PC: 3列、タブレット: 2列、モバイル: 1列）

### 4. サービス特徴セクション (`ServiceFeaturesSection.tsx`)

**機能:**

#### 3ステップ説明

1. 業者を探す（Searchアイコン）
2. カートに追加（ShoppingCartアイコン）
3. 一括問い合わせ（Sendアイコン）

#### メリット訴求

- 完全無料（Shieldアイコン、青色）
- 一括問い合わせ（Zapアイコン、オレンジ色）
- Instagram連携（Instagramアイコン、ピンク色）

**技術的特徴:**

- ステップ間の接続線（デスクトップのみ）
- グラデーション背景
- アニメーション付きホバーエフェクト

### 5. 料金相場セクション (`PricingSection.tsx`)

**機能:**

- 6業種の価格帯表示
  - リフォーム: 10万円〜500万円
  - 新築: 2,000万円〜5,000万円
  - 塗装: 50万円〜200万円
  - 外構: 30万円〜300万円
  - 飲食店改装: 100万円〜1,000万円
  - オフィス改装: 50万円〜500万円
- 各カードに注意事項記載
- 人気カテゴリのバッジ表示
- 見積もり依頼CTA

**技術的特徴:**

- 価格帯の視覚的強調
- 人気バッジ（TrendingUpアイコン）
- グリッドレイアウト（PC: 3列、モバイル: 1列）
